Parameters
Predict()is a function to return overpasses with a TLE provided Predict requires 16 inputs in this order:
  1. Sgp4 class - An object to initialise Simplified General Pertubations 4 operations
  2. Integer - The number of overpass predictions to be returned for a satellite
  3. Unsigned Long - The unixtime of when to start returning overpass predctions, will also return epoch time converted as well
  4. Integer - The timezone to return the times for predictions under. It is ahead by 1 hour. For instance if GMT +8 (Singapore) timezone is needed, 7 would be the input
  5. Double - Latitude of coordinates, use fracPart() to return 4 decimal places
  6. Double - Longitude of coordinates, use fracPart() to return 4 decimal places
  7. Integer - Altitude of coordinates, to nearest meter
  8. const char* - satname, not important or required to return a value
  9. char* - 1st line of the two line element data
  10. char* - 2nd line of the two line element data. Ensure that both lines correspond to the same satellite catalog number and timing
  11. Integer - year placeholder for epoch and Julian date operations
  12. Integer - month placeholder
  13. Integer - day placeholder
  14. Integer - hour placeholder
  15. Integer - minute placeholder
  16. Double - seconds placeholder
